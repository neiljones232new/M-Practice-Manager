# M Practice Manager

Modern client CRM for professional practices (accounting, tax, payroll) with AI-powered assistance.

## Core Features

- **Client Management**: Comprehensive client records with Companies House integration
- **Service & Fee Tracking**: Manage services, frequencies, and annualized fees
- **Task & Deadline Management**: Track work items and compliance deadlines
- **Document Management**: Secure file storage with tagging and search
- **M Assist**: AI-powered insights and automation using OpenAI
- **Compliance Tracking**: Automated filing deadline monitoring
- **Calendar Integration**: Schedule appointments and track deadlines
- **Report Generation**: HTML and PDF client reports with Handlebars templates

## Design Philosophy

- **Modern light theme** with lavender/purple primary branding and subtle gradients
- Glass-morphic UI elements with backdrop blur effects and elegant shadows
- Professional aesthetic suitable for accounting and tax practices
- File-based JSON storage for development (optional PostgreSQL for production)
- Modular architecture with clear separation of concerns
- API-first design with comprehensive Swagger documentation
- Security-focused with JWT authentication and role-based access control

## Target Users

Professional practices (accountants, tax advisors, payroll services) managing multiple clients with compliance requirements.
