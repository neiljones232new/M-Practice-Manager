Overview

The MDJ UI Kit defines a cohesive dark-gold hybrid interface for all functional pages in the MDJ Practice Manager system. It unifies layout, typography, and styling across Dashboard, Clients, Services, Companies House, Tasks, Documents, Audit, Calendar, People, and Settings.

  • Single responsive layout wrapper (MDJLayout)
  • Reusable card container (MDJCard)
  • Auto-responsive grid (MDJSectionGrid)
  • Global color & font variables (mdj-theme.css)

Setup Instructions

  1. Create directories: 
    /components/mdj-ui/
  ├── MDJLayout.tsx
  ├── mdj-layout.module.css
  ├── MDJCard.tsx
  ├── mdj-card.module.css
  ├── MDJSectionGrid.tsx
  ├── mdj-section-grid.module.css
/styles/
  └── mdj-theme.css
  2. Import the global theme in _app.tsx: 
    import '../styles/mdj-theme.css';
  3. Use components in any page: 
    
import { MDJLayout } from '@/components/mdj-ui/MDJLayout';
import { MDJCard } from '@/components/mdj-ui/MDJCard';
import { MDJSectionGrid } from '@/components/mdj-ui/MDJSectionGrid';

export default function Dashboard() {
  return (
    <MDJLayout title="Dashboard">
      <MDJSectionGrid>
        <MDJCard title="Active Clients">
          <p>42 active clients</p>
        </MDJCard>
        <MDJCard title="Tasks Due">
          <p>15 tasks this week</p>
        </MDJCard>
      </MDJSectionGrid>
    </MDJLayout>
  );
}

MDJLayout Component

The unified layout with dark sidebar and header plus light content area. Handles responsive behavior, mobile collapse, and an optional search or breadcrumb slot.

File

/components/mdj-ui/MDJLayout.tsx

// see repo for full code (includes sidebar + header)
// props: title, actions, searchBar, breadcrumb
<MDJLayout
  title="Clients"
  searchBar={<input className="input-dark" placeholder="Search clients..." />}
  actions={<button className="btn-gold">Add Client</button>}
>
  {/* page content */}
</MDJLayout>

Styling

mdj-layout.module.css defines the hybrid gold theme and responsive sidebar.

MDJCard Component

A reusable card wrapper for sections and panels. Supports titles, subtitles, and actions.

Usage Example


<MDJCard title="Upcoming Tasks" subtitle="Next 7 days" actions={<button className="btn-outline-gold">View All</button>}>
  <ul>
    <li>VAT return due for ABC Ltd</li>
    <li>Accounts review meeting</li>
  </ul>
</MDJCard>

MDJSectionGrid Component

Responsive container that arranges cards and sections evenly. Defaults to 3 columns but automatically wraps based on screen width.

Usage


<MDJSectionGrid minWidth="300px">
  <MDJCard title="Metric A"></MDJCard>
  <MDJCard title="Metric B"></MDJCard>
  <MDJCard title="Metric C"></MDJCard>
</MDJSectionGrid>

mdj-theme.css

Global CSS variable definitions and utility classes for buttons, cards, and inputs.

Highlights

  • --gold → #c8a652
  • .btn-gold → primary button style
  • .input-dark → input field style used in layouts
  • .card → base card styling

:root {
  --gold: #c8a652;
  --border: rgba(200,166,82,0.25);
  --text-light: #f0f0f0;
  --text-dark: #1a1a1a;
}
.btn-gold {
  background: var(--gold);
  color: var(--text-dark);
}
